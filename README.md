# TD CONSULTING - Hiring Portal

A modern job application portal for TD CONSULTING built with React, TypeScript, and Vite.

## Features

- Beautiful, responsive UI with dark theme
- Full-time/Part-time/Remote application form
- Supabase integration for data storage
- Discord webhook notifications
- Form validation with zod
- Portfolio link management

## Tech Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS
- Supabase
- React Hook Form + Zod
- Shadcn/ui components

## Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

```bash
npm install
npm run dev
```

The app will be available at http://localhost:8080

## Environment Variables

Create a `.env` file with:

```
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
VITE_DISCORD_WEBHOOK_URL=your_discord_webhook_url
```
